# LingoPop – AI-powered Language Learning Web App

LingoPop is an AI-powered web application designed to help language learners study vocabulary more efficiently.  
It combines dictionary lookup, AI explanations, pronunciation playback, and flashcard-based review in one lightweight tool.

Live Demo:  
https://aidictionary-tau.vercel.app/

---

## Features

### 🔎 Smart Vocabulary Lookup
Search for words and instantly get:
- AI-generated explanations
- contextual meanings
- example sentences

### 🔊 Pronunciation Playback
Listen to the correct pronunciation of words directly in the browser.

### 🧠 AI-powered Explanations
Each word includes clear explanations to help learners understand usage and meaning.

### 📝 Example Sentences
Automatically generated example sentences show how the word is used in context.

### 📚 Verb Conjugation Support
For verbs, the application displays conjugation references to support grammar learning.

### ⭐ Personal Vocabulary Notebook
Users can bookmark words and build a personal vocabulary collection.

### 🎴 Flashcard Review
Saved words can be reviewed using flashcard-style learning to reinforce memory.

### 🌍 Multi-language Learning
Supports learning across multiple languages such as:
- Portuguese
- English

### 📱 Cross-device Access
The app is deployed online and can be used on both:
- Mobile devices
- Desktop browsers

---

## Motivation

While learning Portuguese, I found that switching between multiple dictionary apps slowed down the study process.  
Existing tools often lacked context such as example sentences or verb conjugations.

LingoPop was created to integrate these features into a single lightweight learning tool.
